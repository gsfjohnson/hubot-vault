hubot-vault
===========

Manage vault via hubot.

## Installation

### Update the files to include the hubot-vault module:

#### package.json
    ...
    "dependencies": {
      ...
      "hubot-vault": ">= 0.0.1"
      ...
    },
    ...

#### external-scripts.json
    [...,"hubot-vault"]

Run `npm install` to install hubot-vault and dependencies.
